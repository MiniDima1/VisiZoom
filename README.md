# VisiZoom

VisiZoom - современное приложение для просмотра изображений. Минимализм и плавность - основа проекта.

## Возможности
- Быстрое открытие изображений (PNG, JPG, BMP, GIF, ICO, WEBP, SVG и другие )
- Перемотка фото клавишами: Левой стрелкой / Правой стрелкой
- Плавное масштабирование колёсиком мыши (CTRL+СКМ)
- Перемещение изображения зажатием ЛКМ
- Тёмная тема интерфейса

## Установка
1. Скачайте последнюю версию из раздела **Releases**.
2. Запустите установщик и следуйте инструкциям.

> **Примечание:** Чтобы автоматически создать ассоциации с поддерживаемыми форматами, запускайте установку от имени администратора.

## Требования
- Windows 10 / 11
