# VisiZoom

VisiZoom - современное приложение для просмотра изображений. Минимализм и плавность - основа проекта.

## Возможности
- Быстрое открытие изображений (PNG, JPG, JPEG, BMP, GIF, ICO)
- Плавное масштабирование колёсиком мыши
- Перемещение изображения зажатием ЛКМ
- Тёмная тема интерфейса

## Установка
1. Скачайте последнюю версию из раздела **Releases**.
2. Запустите установщик и следуйте инструкциям.

> **Примечание:** Чтобы автоматически создать ассоциации с поддерживаемыми файлами, запускайте установку от имени администратора.

## Требования
- Windows 10 / 11
